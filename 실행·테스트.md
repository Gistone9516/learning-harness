# 학습 프레임워크 실행·테스트 가이드

---

## ① 브라우저에서 열기

- `엔진/app/index.html` 파일 더블클릭 → 브라우저에서 직접 열림 (`file://` 프로토콜)
- **권장 브라우저**: Chrome 또는 Edge (Web Speech API 지원 필요)
  - Firefox: 대부분 동작하나 TTS·음성인식 미지원
  - Safari: 부분 지원
- **인터넷 연결 필수**: CDN 의존 라이브러리 로드 필요
  - KaTeX(수식), CodeMirror(코드 편집기), Pyodide(Python 실행), Univer(스프레드시트), React, RxJS
  - 오프라인 환경 = 플러그인 UI 렌더 불가

## ② 플러그인별 새 기능 확인 포인트

### 카드퀴즈 (`card-quiz`)
- 목업 카드 6개 로드됨 (`_목업카드.js`)
- **judge 선택지 카드** (`comp1-ss-judge-0001`): 카드 앞면에 선택지 버튼 4개 렌더 확인
  - 정답 클릭 후 뒷면에 `back.why` 정답 근거 텍스트 표시 여부 확인
- **cloze 빈칸** (`comp1-ss-cloze-0001`): 빈칸 입력창 렌더 확인
- **recall_seq 순서** (`comp1-ss-seq-0001`): 배열 입력 UI 확인
- **self 자기평가** (`comp1-ss-self-0001`): O/X 버튼 확인

### 코딩 (`coding`)
- `_example` 6개 + `_목업` 5개 = 총 11개 액티비티
- **전체 케이스 결과표**: 문제 채점 후 케이스별 pass/fail 목록 확인 (`cod-mock-002` = 5케이스)
- **float 허용오차**: `cod-mock-002` stdout-float-tol 주석 확인, `cod-mock-003`(numpy/round) 실행
- **해설 패널**: 정답/오답 후 `back.solution`·`back.explanation` 영역 렌더 확인
- **패키지 허용**: `cod-mock-005` → `allowed_packages: ["numpy"]` 선언됨 (Pyodide micropip 로드 테스트)

### 엑셀 (`excel`)
- `_example` 6개 + `_목업` 4개 = 총 10개 액티비티
- **해설 패널**: 채점 후 `back.explanation` 영역 표시 확인
- **float 허용오차**: `xls-avg-001` (AVERAGE → float-tol 적용)
- **다중 셀 결과표** (`xls-sumif-001`): `target_cells: ['B10','B11','B12']` = 셀 3개 각각 pass/fail 표 렌더 확인

### 영어 (`english`)
- `_example` 6개 + `_목업` 6개 = 총 12개 액티비티
- **받아쓰기 임계** (`eng-lst-dictation-001`): `dictation_threshold: 0.75` 재정의 — 75% 이상 정답 처리
- **TTS 속도 조절**: 영어 카드 재생 버튼 클릭 시 속도 슬라이더 동작 확인 (Web Speech API)
- **writing 히스토리** (`eng-wrt-email-001`): LLM 루브릭 채점 → API 키 없으면 `verdict=pending` graceful
- **발음 채점** (`eng-spk-th-sound-001`): 마이크 권한 허용 후 음성인식 → 키 없으면 `verdict=pending` graceful

### AWS (`aws`)
- `_example` 11개 + `_목업` 1개 = 총 12개 액티비티 (중복 제거 완료)
- **멀티 과제 사이드바**: 사이드바에서 12개 과제 목록 확인
- **체크포인트 체크박스** (`aws-sns-create-topic-001`): SNS 토픽 생성 과제 — 체크포인트 항목 체크박스 동작 확인

## ③ Node.js 단위 테스트 실행

```
node 엔진/plugins/card-quiz/score.test.js
node 엔진/plugins/coding/__tests__/grade.test.js
node 엔진/plugins/excel/_test_scoring.js
node 엔진/plugins/english/test_english.js
node 엔진/app/boot_harness.js
```

- Node.js 설치 필요 (v18 이상 권장)
- 명령은 프로젝트 루트(`학습 프레임워크 제작/`)에서 실행
- 전부 브라우저·인프라 없이 로컬 실행 가능

## ④ 브라우저/인프라가 필요한 항목 (사용자 몫)

| 기능 | 필요 조건 |
|---|---|
| Python 코드 실행 (Pyodide) | 브라우저 + 인터넷 (CDN 로드) |
| numpy·matplotlib 패키지 | Pyodide micropip 자동 설치 (시간 소요) |
| 스프레드시트 UI (Univer) | 브라우저 + 인터넷 |
| 수식 렌더 (KaTeX) | 브라우저 + 인터넷 |
| TTS·발음채점 (Web Speech) | Chrome/Edge + 마이크 권한 |
| Writing LLM 루브릭 채점 | BYO API 키 (키 없으면 pending 반환) |
| 음성인식 채점 | Azure Speech 키 또는 Web Speech API |
| AWS 실습 실제 실행 | AWS 계정 + IAM 권한 + Docker(SAM 등) |
