# Changelog

All notable changes to this harness kit are documented here. The format follows
[Keep a Changelog](https://keepachangelog.com), and this project adheres to
[Semantic Versioning](https://semver.org).

## [0.1.0] - 2026-06-14
### Added
- Initial harness kit: 57 self-contained, copy-style Discord capability files
  across output / live / interaction / slash / channels / media / automation /
  meta (mainstream) and community (server management).
- `README.md` with usage rules and a two-tier capability index.
- `index.json` machine-readable manifest plus `build_index.py` generator.
- Per-view timeout cleanup, public-API type hints, and an `INTENTS` header field.
- MIT license. Built with AI (Claude).
