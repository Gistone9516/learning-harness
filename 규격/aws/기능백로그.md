## low

|---|---|---|---|---|
| **VPC·보안그룹 과제** | VPC 생성·서브넷·보안그룹 과제. `ec2 describe_vpcs/describe_security_groups` 채점 | backend | M | mid→low 재배치. EC2 Full 지원 여부 M 불확실성 잔존. LocalStack EC2 = mock 수준 [도메인 능력 항목 직접 인용]. config-equals 신뢰도 낮음 |
| **CloudFormation 과제** | CFN 스택 배포 과제. `cloudformation:describe_stacks` `resource-exists` check로 `CREATE_COMPLETE` 검증 (별도 `cfn-deploy` 타입 불필요 — 기존 체계 흡수) | backend | M | mid→low 재배치. [review: 별도 타입 설계 과장, 기존 `resource-exists`로 흡수 가능. 입문 흐름 단절 우려. 기초 과제 데이터 완성 후순] |
| **IAM policy-contains 채점** | `grade.py`에 `policy-contains` check type. `list_attached_role_policies`로 특정 정책 연결 검증 | backend | S | 파킹 항목 재평가. 최소 권한 원칙 학습 가치 있음. 단 기초 과제 세트 완성 후 진입 |
| **LLM 기반 해설 강화** (BYOK) | 채점 실패 시 checks 결과 + prompt → LLM(OpenAI/Anthropic BYOK) → 자연어 해설. 키 없으면 graceful(기존 텍스트 피드백 유지) | byok | M | 키 불요 기능 우선. back.explanation 완성 후 보완 역할. 사용자 과금 발생 |
