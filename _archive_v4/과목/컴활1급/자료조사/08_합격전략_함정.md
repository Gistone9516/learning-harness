## 2. 시험 구조 요약

### 필기
- 3과목: 컴퓨터 일반(20문) / 스프레드시트 일반(20문) / 데이터베이스 일반(20문)
- 60분, 객관식
- 합격 기준: 과목당 40점 이상 + 평균 60점 이상

### 실기
- 2과목: 스프레드시트 실무 / 데이터베이스 실무
- 각 45분 (총 90분)
- 합격 기준: **두 과목 모두** 70점 이상 (어느 한 과목이라도 70점 미달 → 불합격)

---

## 3. 필기 전략

### 3-1. 과목별 우선순위 및 특징

1. **컴퓨터 일반** (★★☆) — 기출문제 반복만으로 합격 수준 달성 가능. 운영체제·네트워크·저장장치·정보보호 중점.
2. **스프레드시트 일반** (★★★ 최주의) — 기출 비중 낮고 변형·신유형 비중 높아 과락 유발 빈도 1위 과목. 함수 문제는 기출만으로 손 못 댈 가능성 높음. 기초 개념 + 기출 병행 필수.
3. **데이터베이스 일반** (★★☆) — 기본 성향은 스프레드시트와 유사하나 상대적으로 덜 어렵게 출제되는 편. 기본키·외래키·SQL SELECT문 중점.

### 3-2. 과락 방지 전략
- 과락 기준: 한 과목 8문제(40점) 미만 → 전체 불합격
- 전략: 평균 80점 이상이면 과락 불발생. 한 과목 100점이면 나머지 두 과목은 과락만 면해도 합격.
- **버리기 원칙**: 완전히 모르는 개념보다 자주 나오는 유형 70% 확보가 우선. 스프레드시트는 함수 완전 포기 금지(과락 위험).

### 3-3. 시간배분 (7일 단기 플랜 기준)
- 1~3일: 3과목 핵심 개념 정리 + 유형 파악 (하루 2~3시간)
- 4~7일: 최근 5개년 기출문제 반복 풀이

### 3-4. 필기 합격 후
- 필기 합격일로부터 **2년 이내** 실기 응시해야 유효

## 4. 실기 전략

### 4-1. 스프레드시트 실무 (Excel) — 45분

**배점 구성 (참고)**
- 기본작업: 15점 (셀 서식, 조건부 서식, 데이터 유효성 등)
- 계산작업: 30점 (함수 — 최고 배점)
- 분석작업: 20점 (피벗테이블, 데이터 도구 등)
- 기타작업: 35점 (차트 10점 + 매크로 10점 + 프로시저/VBA 15점)

**전략**
- 기본작업·차트·매크로(55점 상당)를 최대한 안정적으로 확보한 뒤, 계산작업(함수) 부분점수 극대화
- 함수는 복합 중첩 함수가 핵심 — IF, VLOOKUP, INDEX/MATCH, SUMIF, COUNTIF, ROUND 등 숙달 필수
- 45분 내 완성 속도 체득이 목표 — 막히는 문제는 과감히 패스, 후반에 재시도

**난이도**: 액세스보다 엑셀이 더 어려움(2024 개정 이후 유형 다양화)

### 4-2. 데이터베이스 실무 (Access) — 45분

**전략**
- 암기 분량이 방대 — 문제 지문 읽는 순간 손이 먼저 갈 정도로 반복 숙달
- DB구축(테이블/관계) → 입력/수정 폼 → 쿼리 → 보고서 순서로 학습
- 쿼리는 SQL보다 디자인 보기 우선 숙달
- 액세스는 일부 회차에서 상대적으로 쉽게 출제되는 경향 있음("복불복" 성격)

## 5. 감점·함정 TOP 목록

> 총 14항목

1. **파일 저장 미처리** — 시험 종료 전 반드시 저장(Ctrl+S). 저장 안 하면 작업 내용 0점 처리.
2. **파일 확장자 오류** — 매크로 포함 파일은 `.xlsm`으로 저장 필수. `.xlsx`로 저장 시 매크로 소실 → 해당 작업 0점.
3. **시트 순서 변경** — 기본작업1-기본작업2-계산작업-분석작업1-분석작업2-기타작업1-기타작업2 순서 유지 필수. 시나리오·피벗 요약 시트 삽입 위치 실수 → 실격 처리.
4. **셀 서식 코드 오류** — 숫자 서식 코드(#,##0 / 0.00% 등) 한 글자 오기 → 해당 항목 감점.
5. **반올림 함수 인수 오류** — ROUND(값, 자릿수): 자릿수 1=소수 첫째, 0=정수, -1=십의 자리. 혼동 빈번.
6. **ROUND vs 표시 형식 혼용** — 서식으로 소수점 자리 맞추면 값 자체는 반올림 안 됨 → 계산 오류 원인.
7. **매크로 불필요 동작 기록** — 매크로 녹화 중 문제 풀이와 무관한 동작(클릭·스크롤 등) 그대로 기록 → 감점.
8. **액세스 필드 삭제** — 임의 필드 삭제 시 팝업 없이 자동 저장, 복구 불가 → 해당 테이블 작업 전체 감점.
9. **액세스 임의 필드 생성** — 문제에 없는 필드 생성 → 감점. 반드시 삼가야 함.
10. **데이터 통합 범위 추가/누락** — 통합 원본 범위를 잘못 지정 → 0점 처리 사례 존재.
11. **조건부 서식 조건 누락** — 조건식 AND/OR 논리 오류, 조건 개수 누락 → 부분 감점.
12. **피벗테이블 필드 설정 오류** — 반올림·빈셀·요약 함수(합계/평균 혼동) 필드 설정 확인 필수.
13. **시험장 환경 미확인** — 시험 시작 전 Excel 환경 설정·화면 배치 확인 안 할 경우 예기치 못한 오작동.
14. **필기 합격 유효기간 만료** — 필기 합격 후 2년 초과 시 실기 응시 불가(처음부터 재응시 필요).

## 6. 부분점수 구조

- **엑셀**: 계산작업·분석작업 등 일부 항목에서 부분점수 존재. 함수 인수가 일부 틀려도 구조가 맞으면 부분 인정 가능.
- **액세스**: DB구축·쿼리·폼·보고서 각 항목별 세부 채점. 항목 내 세부 요소 일부 누락 시 감점.
- **매크로·VBA**: 실행 결과가 맞아야 점수 인정 — 중간 오류로 실행 불가 시 0점 처리 가능성 높음.
- **공통**: 문제 조건을 완전히 충족해야 만점; 일부 충족은 항목별로 부분점수 적용 여부 상이.

## 7. 독학 로드맵

### 7-1. 권장 순서

```
[STEP 1] 필기 준비 (1~2주)
  → CBT 기출문제 중심 (시나공/이기적 600~1200문제)
  → 취약과목 개념 보충 (스프레드시트 함수 개념 포함)
  → 필기 합격

[STEP 2] 실기 준비 (4~8주)
  → 엑셀 기초 함수/기능 인강 1회독 (유동균·정익종 등 전문 강의 권장)
  → 액세스 전 기능 암기 + 반복 실습
  → 기출문제 반복 (2024~2025 기출 중심)
  → 45분 타임어택 연습

[STEP 3] 마무리 (D-1주)
  → 시험 환경(파일저장·확장자·시트순서) 체크리스트 점검
  → 감점 함정 TOP 재확인
  → 최근 기출 1회분 실전 모의
```

### 7-2. 대상별 권장 기간

| 대상 | 필기 | 실기 | 총계 |
|------|------|------|------|
| 노베이스 | 1~2주 | 6~8주 | 약 2~3개월 |
| 엑셀 기초 있는 대학생 | 1주 | 5~7주 | 약 6~10주 |
| 2급 보유자 | 3~5일 | 3~5주 | 약 4~6주 |
| 취준생(집중 투자) | 1주 | 3~4주 | 최소 4주 |

### 7-3. 하루 권장 분량
- 필기: 하루 2~3시간 / 기출 50~100문제 반복
- 실기: 하루 2~3시간 실습 위주 / 타임어택 1세트(90분) 주 2회 이상

## 8. 응시 팁

1. **상시 검정 활용** — 명절·선거공휴일 제외 거의 매일 시험. 필기 합격 직후 빠르게 실기 일정 잡기 권장.
2. **실기는 과목 모두 70점 이상 필수** — 한 과목 점수가 높아도 다른 과목 70점 미달이면 불합격.
3. **액세스 '복불복' 활용** — 쉬운 회차를 노리는 전략도 가능 (상시 특성상 회차별 난이도 편차 존재).
4. **인강 필수 추천** — 2024 개정 이후 독학 독자 학습은 합격률 극히 낮음. 전문 강의(유동균·정익종 등) 병행 권장.
5. **3개월 여유 설정** — 단기 속성보다 정석 학습이 누적 응시 비용·시간 합산 시 유리.
