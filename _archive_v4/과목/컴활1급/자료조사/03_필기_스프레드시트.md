# 컴활1급 필기 — 스프레드시트 일반

> 기준: MS Office 2021 / 2024-2026 출제기준 (대한상공회의소)
> 총 20문항, 과락 40점, 평균 60점 이상 합격

---

## 1. 출제기준 트리 (대단원 → 중단원 → 세부항목 전수)

### 대단원 1. 엑셀 일반 (화면 구성·워크시트·통합 문서 관리)

#### 1-1 화면 구성
- 빠른 실행 도구 모음 (바로 가기 키 자동 부여)
- 리본 메뉴 (탭·그룹·명령 구조 / 탭 종류: 홈, 삽입, 페이지 레이아웃, 수식, 데이터, 검토, 보기, 개발 도구, 추가 기능, 팀)
- 이름 상자 (셀 주소·차트·도형 이름 표시)
- 수식 입력줄 (수식 그대로 표시)
- 상태 표시줄 (평균·개수·숫자 셀 수·최소값·최대값·합계 자동 계산)

#### 1-2 워크시트 관리
- 구성: 가로 16,384열 × 세로 1,048,576행
- 이름 변경 (최대 31자, `* / : ? [ ]` 사용 불가)
- 이동·복사·삽입·삭제 (삭제된 시트는 되살리기 불가)
- 다중 시트 선택 (연속: Shift·클릭 / 비연속: Ctrl·클릭)
- 그룹 상태 제한 사항: 도형·차트 삽입, 정렬·필터 불가
- 시트 보호 / 통합 문서 보호 (구조·창 보호, 암호 설정)
- 통합 문서 공유 (조건부 서식·차트·시나리오 등 추가·변경 불가)

#### 1-3 통합 문서 관리
- 파일 형식: `.xlsx` (기본) / `.xlsm` (매크로 포함) / `.xltx` (서식) / `.xltm` (매크로 서식) / `.xlsb` (바이너리) / `.xlk` (백업) / `.csv` / `.txt` / `.xls` (구버전) 등
- 서식 파일: 기본 저장 위치 `Templates` 폴더, 기본 서식 파일은 XLStart 폴더에 `sheet`·`book`으로 저장
- 열기 암호 (최대 255자) / 쓰기 암호 (최대 15자) / 읽기 전용 권장

### 대단원 2. 입력 및 편집

#### 2-1 데이터 형식 및 입력
- 문자 데이터: 기본 왼쪽 정렬 / 숫자 앞 문자 접두어(`'`) 입력 시 문자 처리
- 숫자 데이터: 기본 오른쪽 정렬 / 분수 `0 1/2` 형태 / 너비 초과 시 지수 표시
- 날짜 구분: `-` 또는 `/` / 연도 2자리: 00~29→2000년대, 30~99→1900년대
- 시간 구분: `:` / 기본 24시각제 / AM·PM으로 12시각제 전환
- 날짜 일련번호: 1900-01-01 = 1 / 시간 소수(자정=0.0, 정오=0.5)
- 오늘 날짜: `Ctrl+;` / 현재 시간: `Ctrl+Shift+;`
- 여러 줄 입력: `Alt+Enter` / 여러 셀 동시 입력: `Ctrl+Enter`
- 셀 내용 자동 완성: 문자 데이터만 적용, 숫자·날짜·시간 제외
- 윗주: 문자 데이터에만 삽입 가능, 데이터 삭제 시 윗주도 삭제
- 메모: 빨간 삼각형(▲) 표시 / `Shift+F2` 삽입 / 항상 표시·인쇄 설정 가능

#### 2-2 채우기 핸들
- 숫자(한 셀): 드래그→복사 / Ctrl+드래그→1씩 증가
- 숫자(두 셀): 두 값 차이만큼 증감
- 문자: 사용자 지정 목록 순서대로 반복
- 혼합(문자+숫자): 숫자만 증감
- 날짜(한 셀): 1일 단위 증가 / 두 셀: 차이만큼 년·월·일 단위 증가

#### 2-3 선택하여 붙여넣기
- 모두·수식·값·서식·메모·유효성 검사·열 너비·수식과 숫자 서식·행열 바꿈·연결하여 붙여넣기
- 연산: 더하기·빼기·곱하기·나누기
- **잘라내기 후에는 사용 불가 (복사 후에만 가능)**

#### 2-4 셀 서식
- 사용자 지정 서식: 양수·음수·0·텍스트 4구역 (`#,### ; [빨강](#,###) ; 0.00 ; @"님"`)
- 서식 코드: `#`(유효 자릿수만) / `0`(불필요 0 채움) / `?`(공백으로 정렬) / `,`(천 단위)
- `%`(×100+%) / `@`(문자 위치) / `*`(반복 채움) / `_`(공백)
- 날짜: `mmm`(Jan~Dec) / `mmmm`(January~December)
- 요일: `ddd`(Sun~Sat) / `dddd`(Sunday~Saturday)
- 시간: `hh`(00~23) / `[h]`(경과 시간) / `mm`(분) / `[m]`(경과 분)

#### 2-5 조건부 서식
- 수식으로 지정 시 반드시 `=` 입력
- 값 변경으로 조건 불만족 시 서식 자동 해제
- 데이터 막대·색조·아이콘 지원
- 우선 순위 높은 규칙이 충돌 시 적용
- 다른 워크시트·통합 문서 참조 조건 불가
- 규칙 관리자: [홈] → [스타일] → [조건부 서식] → [규칙 관리]

#### 2-6 데이터 유효성 검사
- 제한 대상: 모든 값·정수·소수점·목록·날짜·시간·텍스트 길이·사용자 지정
- 제한 방법: 제외 범위·=·<>·>·<·>=·<=
- 설명 메시지·오류 메시지 지정 (오류 스타일: 경고·중지·정보)
- IME 모드 지정 가능

### 대단원 3. 수식과 함수

#### 3-1 수식·참조
- 수식 시작: `=` / `+` / `-`
- 셀에 결과값, 수식 입력줄에 수식 표시
- 수식→상수 변환: 수식 입력 후 Enter 전 F9
- 입력된 수식 보기: `Ctrl+~`
- **상대 참조** (A1): 수식 위치 이동 시 참조 변경
- **절대 참조** ($A$1): 위치 무관 고정
- **혼합 참조** ($A1 열 고정 / A$1 행 고정)
- 다른 시트 참조: `=Sheet1!A5` (시트명에 특수문자 포함 시 작은따옴표)
- 3차원 참조: `=SUM(Sheet1:Sheet3!B2)` (배열 수식에는 사용 불가)
- 외부 통합 문서 참조: `='C:\[매출현황]Sheet4'` (경로명 작은따옴표)

#### 3-2 오류 메시지
| 오류 | 원인 |
|------|------|
| `#####` | 셀 너비 부족 / 음수 날짜·시간 |
| `#DIV/0!` | 0 또는 빈 셀로 나눔 |
| `#N/A` | 사용할 수 없는 값 |
| `#NAME?` | 인식 불가 텍스트 |
| `#NULL!` | 교차하지 않는 영역 교점 |
| `#NUM!` | 숫자 범위 초과 |
| `#REF!` | 유효하지 않은 셀 참조 |
| `#VALUE!` | 잘못된 인수·피연산자 |

#### 3-3 이름 정의
- 첫 문자: 문자(한글·영문) / 밑줄(`_`) / 역슬래시(`\`)
- 공백 포함 불가 / 대소문자 구분 안 함 / 최대 255자
- 동일 통합 문서 내 중복 불가
- 정의된 이름은 절대 참조 방식으로 사용

#### 3-4 배열 수식 및 배열 상수
- 입력: `Ctrl+Shift+Enter` → 자동으로 `{ }` 추가
- 배열 수식 일부만 수정 불가
- 배열 상수: 숫자·텍스트·논리값·오류값 사용 가능 / `$`·괄호·`%`·셀 참조·길이 다른 행열 불가
- 열 구분: `,` / 행 구분: `;`
- 3차원 참조 사용 불가

#### 3-5 배열 수식 활용 패턴
- 조건 1개, 개수: `{=SUM((조건)*1)}` / `{=SUM(IF(조건,1))}` / `{=COUNT(IF(조건,1))}`
- 조건 2개, 개수: `{=SUM((조건1)*(조건2)*1)}`
- 조건 1개, 합계: `{=SUM((조건)*(합계범위))}`
- 조건 1개, 평균: `{=AVERAGE(IF(조건,범위))}`

| `DMAX` / `DMIN` | 조건 최대/최소 |
| `DPRODUCT` | 조건 곱 |
| `DSTDEV` / `DVAR` | 조건 표준 편차 / 분산 |
| `DGET` | 조건 단일값 |

#### 3-13 재무 함수

| 함수 | 설명 |
|------|------|
| `FV(이자,기간,금액,현재가치,납입시점)` | 미래가치 |
| `PV(이자,기간,금액,미래가치,납입시점)` | 현재가치 |

| `SLN(취득액,잔존가치,수명년수)` | 정액법 감가상각 |
| `DB(취득액,잔존가치,수명년수,기간,월수)` | 정율법 감가상각 |

> 납입시점: 0(생략)=기말 / 1=기초

  - `Private`: 모듈 처음 선언, 해당 모듈 전체

#### 8-5 VBA 제어 문법

**If ~ Then**
```vba
If 조건식 Then
    실행문
Else
    실행문
End If
```

**Select Case**
```vba
Select Case 변수
    Case 조건1: 실행문
    Case Else: 실행문
End Select
```

**For ~ Next**
```vba
For 변수 = 시작 To 끝 [Step 증감값]
    실행문
Next [변수]
```

**Do While ~ Loop / Do Until ~ Loop**
```vba
Do While 조건식   ' 조건 참인 동안 반복
    실행문
Loop

Do Until 조건식  ' 조건 참이 될 때까지 반복
    실행문
Loop
```

**기타**: `Exit For` / `Exit Do` / `Exit Sub` / `Exit Function`

## 2. 단원별 출제 경향 및 함정 포인트

### 대단원 1 — 엑셀 일반
- **함정**: 시트 보호 ≠ 통합 문서 보호 / 공유 통합 문서 제한 항목 암기 (조건부 서식·차트 추가 불가)
- 실행 취소 불가 항목: 시트 이름 변경·삽입·삭제·복사·이동, 틀 고정, 창 숨기기

### 대단원 2 — 입력 및 편집
- **날짜 두 자리 연도**: 00~29=2000년대, 30~99=1900년대 자주 출제
- **선택하여 붙여넣기**: 잘라내기 후 불가 함정
- **조건부 서식 행 전체 서식**: 수식에 비교 열 앞에 `$` 붙이는 혼합 참조 패턴 빈출
- **자동 채우기**: 숫자 한 셀(드래그=복사, Ctrl+드래그=증가) 혼동 주의

### 대단원 3 — 수식과 함수
- **함수 인수 순서** 줄세우기 문제 빈출 (VLOOKUP·HLOOKUP 옵션 TRUE/FALSE 의미)
- **RANK.EQ vs RANK.AVG** 차이 명확히
- **FIND vs SEARCH**: 대소문자 구분·와일드카드 사용 여부 반대
- **배열 수식**: Ctrl+Shift+Enter / { } 자동 입력 / 일부 수정 불가
- **재무 함수 납입시점**: 0=기말, 1=기초
- **이름 정의**: 첫 글자 제한, 절대 참조 방식, 공백 불가

### 대단원 4 — 차트
- **추세선 추가 불가 차트** 암기: 3차원·방사형·원형·도넛형·표면형
- **혼합형 차트 불가** 암기: 3차원·주식형·거품형·표면형·원통형·피라미드형
- **원형 차트**: 계열 1개, 백분율 레이블 가능, 축 없음
- **오차 막대**: 3차원 불가, 분산형·거품형만 가로+세로 모두 가능

### 대단원 5 — 출력
- **페이지 설정 탭 4개** 항목별 귀속 구분 (시트 탭 항목 vs 여백 탭 항목)
- **자동 나누기=점선, 수동 나누기=실선**
- **페이지 레이아웃 보기**: 페이지 구분선 마우스 조절 불가
- **확대/축소**: 인쇄에 미적용

### 대단원 6 — 데이터 관리
- **정렬 순서 오름차순**: 숫자>문자>논리>오류>빈 셀 / 문자 내 특수>영문(소>대)>한글
- **자동 필터 상위 10**: 숫자 필드만 / 두 필드 이상 조건=AND
- **고급 필터 조건**: 같은 행=AND, 다른 행=OR / 수식 조건 시 필드명 다르게
- **부분합**: 사전 정렬 필수 / 제거 시 윤곽도 제거
- **피벗 테이블**: 값 영역 수정 불가 / 피벗 차트 생성 시 피벗 테이블 자동 생성

### 대단원 7 — 가상 분석
- **시나리오 최대 변경 셀 32개** / 시나리오 삭제해도 요약 보고서 유지
- **목표값 찾기**: 변수 1개만 / 수식 셀 필수
- **데이터 표**: 결과 일부 수정 불가

### 대단원 8 — 매크로와 VBA
- **매크로 바로 가기 키**: 엑셀 기존 단축키와 충돌 시 매크로 우선
- **PERSONAL.XLSB**: XLSTART 폴더, 엑셀 실행 시 항상 자동 열림
- **변수 선언 키워드별 범위** 차이 (Dim/Static/Private/Public)
- **Sub vs Function**: 반환값 유무 구분
- **이벤트 프로시저**: `개체명_이벤트명` 형식
