## 1. 기본작업 (15점)

|---|-----------|----------------|

| 2 | 셀 서식 — 맞춤 | 가로·세로 맞춤, 들여쓰기, 텍스트 줄 바꿈, 병합 |

| 6 | 조건부 서식 | 규칙 설정(값/수식), 데이터 막대, 색조, 아이콘 집합 |

### 1-2. 배점 및 부분점수

- 총 15점; 문항별 통상 **3~5점** (연도·회차별 변동)
- 셀 서식류: 항목 단위 부분 득점 가능 (표시 형식 맞으면 2점, 색 틀리면 −1점 방식)
- 조건부 서식: 규칙 조건 정확해야 득점 (범위 오류·연산자 오류 → 0점)
- 인쇄 설정: 항목별 독립 채점 (머리글 맞음 + 인쇄 영역 오류 → 머리글만 득점)

### 1-3. 자동채점 관점 산출물

| 산출물 | 판정 방법 |
|--------|-----------|
| 셀 NumberFormat 문자열 | 정확 일치 또는 기능 동치 판정 |
| 조건부 서식 규칙(조건·범위·서식) | FormatConditions 컬렉션 비교 |
| 인쇄 영역 문자열 | PageSetup.PrintArea 문자열 비교 |
| 시트 이름·탭 색 | Name, Tab.Color 속성 비교 |
| 셀 병합 영역 | MergeArea 범위 비교 |

### 1-4. 감점 포인트

- 표시 형식 대소문자 구분 없이 비교되나 #과 0 혼용 오류는 감점
- 조건부 서식에서 절대·상대 참조 혼동 → 규칙 전체 오답
- 인쇄 영역에 불필요한 빈 열 포함 → 부분 감점
- 시트 순서 변경 미이행 → 해당 항목 0점

- 각 문항에 함수 1~3개 복합 사용 요구
- 배열수식(Ctrl+Shift+Enter) 지정 요구 포함 가능
- 사용자 정의 함수(VBA Function 프로시저) 작성 요구 포함 가능

#### (2) 논리 함수 — 9개

#### (3) 데이터베이스 함수 — 10개

DAVERAGE, DCOUNT, DCOUNTA, DGET, DMAX, DMIN, DPRODUCT, DSTDEV, DSUM, DVAR

#### (4) 문자열 함수 — 18개

CONCAT, EXACT, FIND, FIXED, LEFT, LEN, LOWER, MID, PROPER, REPLACE, REPT, RIGHT, SEARCH, SUBSTITUTE, TEXT, TRIM, UPPER, VALUE

#### (5) 수학과 삼각함수 — 24개

### 2-3. 배점 및 부분점수

- 문항당 6점; 정답 셀의 값이 기준값과 일치해야 득점
- 복합 함수 문항에서 최종 값만 채점 (중간 수식 오류는 최종값 오류로 연결됨)
- 배열수식 미입력(Ctrl+Shift+Enter 누락) → 0점
- 이름 정의 범위가 수식에 쓰인 경우: 이름 오정의 → 수식 오답 처리
- 반올림 자릿수 오류(ROUND vs ROUNDDOWN 혼용) → 0점

### 2-4. 자동채점 관점 산출물

| 산출물 | 판정 기준 |
|--------|-----------|
| 셀의 계산 결과값 | 기준값과 수치·문자열 완전 일치 |
| 수식 문자열(HasArray) | 배열수식 여부 플래그 확인 |
| 사용자 정의 함수 반환값 | VBA Function 결과 실행 값 비교 |

### 2-5. 빈출 함수 패턴 (출제 빈도 ↑)

- SUMIFS / COUNTIFS / AVERAGEIFS: 다중 조건 집계
- VLOOKUP / XLOOKUP: 참조 조회
- IF / IFS / IFERROR: 조건 분기
- INDEX+MATCH: 동적 참조
- TEXT / LEFT / MID / RIGHT: 문자열 가공
- RANK.EQ / LARGE / SMALL: 순위
- DSUM / DAVERAGE: DB 함수 + 조건 범위
- PMT / PV / FV: 재무 (1~2회차당 1문항 빈도)

### 2-6. 감점 포인트

- 절대참조($) 누락으로 자동 채우기 시 범위 어긋남
- VLOOKUP 4번째 인수 TRUE(근삿값) ↔ FALSE(정확) 혼용
- IFERROR 중첩 위치 오류 (IFERROR 밖에 수식 있어야 할 경우)
- 날짜 함수에서 직렬 번호 vs 날짜 형식 미구분

### 3-2. 배점 및 부분점수

- 총 20점; 통상 3~4문항
- 피벗 테이블: 필드 배치·레이아웃·그룹화 항목별 부분 득점
- 고급 필터: 조건 범위 정확 + 복사 위치 정확 둘 다 맞아야 득점 (일반적으로 전체 or 0)
- 부분합: 기준 필드 + 함수 + 대상 필드 3요소 모두 일치해야 득점
- 시나리오: 시나리오 이름·변경 셀·값 모두 정확해야 요약 포함 득점

### 3-3. 자동채점 관점 산출물

| 산출물 | 판정 방법 |
|--------|-----------|
| 필터 결과 셀 범위 값 | 기준 결과 범위와 값 일치 비교 |
| 피벗 테이블 구조 | PivotFields 배치·DataFields 함수 비교 |
| 시나리오 목록 | Scenarios 컬렉션 이름·값 비교 |
| 데이터 표 결과 범위 | 결과 셀 배열값 비교 |
| 부분합 결과 행 | SUBTOTAL 행 위치·값 비교 |

### 3-4. 감점 포인트

- 고급 필터 조건 범위에 제목 행 누락 → 필터 오작동 → 0점
- 피벗 테이블 값 필드 함수를 합계 대신 개수로 잘못 지정
- 부분합 전 정렬 미이행 → 그룹 분리 오류
- 시나리오 요약 보고서 대신 피벗 보고서 작성 → 부분 감점

- 데이터 범위 오류(불연속 범위 Ctrl 선택 누락 등) → 계열 항목 0점

| SeriesCollection 범위 | 계열 범위 주소 일치 |
| ChartTitle.Text | 문자열 일치 |
| DataLabel 속성 | ShowValue/ShowPercent 플래그 |
| Legend.Position | 위치 코드 비교 |

#### 감점 포인트

- 2차원 ↔ 3차원 차트 혼동
- 묶은 세로 막대 vs 누적 세로 막대 혼동
- 차트 제목 편집 후 Enter 미처리로 편집 모드 잔류

### 4-C. 프로시저 작업 (15점)

#### 세부 유형 전수 열거 (N = 5종)

| # | 세부 유형 | 주요 요구 내용 |
|---|-----------|----------------|
| 1 | Sub 프로시저 | 범위 조작·서식·루프·조건 분기 구현 |
| 2 | Function 프로시저 (사용자 정의 함수) | 계산 로직 구현 → 워크시트에서 호출 |
| 3 | 이벤트 프로시저 | Worksheet_Change, Workbook_Open 등 이벤트 대응 코드 |
| 4 | MsgBox / InputBox | 대화상자 입력·출력 처리 |
| 5 | UserForm | 폼 디자인 + 컨트롤 이벤트 코드 |

#### 배점 및 부분점수

- 3문항 × 5점 = 15점
- 문항 내 요구사항 여러 개: 항목별 부분 득점 가능 (예: 5점 문항에서 2개 항목 각 2.5점)
