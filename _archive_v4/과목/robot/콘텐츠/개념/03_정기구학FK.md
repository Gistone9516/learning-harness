## 정기구학(FK, Forward Kinematics)

정기구학이란 **관절 각도(θ)를 알 때 끝점(end-effector)의 위치를 계산하는 방법**이다.

즉, "관절을 이렇게 돌렸을 때 팔 끝이 어디에 있나?"에 답하는 계산이다.

### 1-링크 로봇 (단순 예)

링크 길이 L₁, 관절 각도 θ₁일 때 끝점 위치:

```
x = L₁ × cos(θ₁)
y = L₁ × sin(θ₁)
```

각도 θ₁이 0°이면 끝점은 오른쪽 방향으로 L₁만큼 뻗어 있다.

### 2-링크 로봇 (FK 핵심)

링크 길이 L₁, L₂, 관절 각도 θ₁, θ₂일 때 끝점 위치:

```
x = L₁ × cos(θ₁) + L₂ × cos(θ₁ + θ₂)
y = L₁ × sin(θ₁) + L₂ × sin(θ₁ + θ₂)
```

**핵심 규칙**: 두 번째 관절의 각도는 첫 번째 관절 각도를 **누적**해서 더한다.

### 직관적 이해

| 상황 | 끝점 위치 변화 |
|------|------|
| θ₁만 증가 | 전체 팔이 θ₁ 방향으로 회전 |
| θ₂만 증가 | 링크2 부분만 추가로 회전 |
| θ₁ = θ₂ = 0° | 끝점 = (L₁+L₂, 0) — 완전히 펼쳐진 상태 |
| θ₁ = 0°, θ₂ = 180° | 끝점 = (L₁-L₂, 0) — 링크2가 접힌 상태 |

### FK vs IK

| 구분 | 입력 | 출력 | 난이도 |
|------|------|------|------|
| FK (정기구학) | 관절 각도 θ | 끝점 위치 (x, y) | 쉬움 (수식 대입) |
| IK (역기구학) | 끝점 위치 (x, y) | 관절 각도 θ | 어려움 (역으로 풀기) |

입문 실습에서는 **FK만** 다룬다.
