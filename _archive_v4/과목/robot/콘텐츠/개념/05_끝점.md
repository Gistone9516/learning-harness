## 끝점(End-Effector)

**끝점(end-effector)** 은 로봇 팔의 **가장 끝에 달린 도구 또는 위치**다. 로봇이 실제로 작업을 수행하는 지점이다.

### 역할

| 맥락 | 끝점의 의미 |
|------|------|
| 용접 로봇 | 용접 토치 끝 |
| 조립 로봇 | 그리퍼(집게) 끝 |
| 학습 시뮬레이터 | 마지막 링크의 끝 좌표 |

### 끝점 좌표

FK 실습에서 끝점 위치는 `(x, y)` 좌표로 표현한다.

2링크 로봇에서:
```
end_x = L₁·cos(θ₁) + L₂·cos(θ₁+θ₂)
end_y = L₁·sin(θ₁) + L₂·sin(θ₁+θ₂)
```

### 목표점과의 거리

실습에서는 끝점을 **목표점(target)** 에 최대한 가깝게 이동시키는 것이 과제다.

```
거리 = sqrt((end_x - target_x)² + (end_y - target_y)²)
```

거리가 **허용 오차(tolerance)** 이하이면 정답으로 판정한다.

### 특이점(Singularity)

특정 자세에서 끝점의 이동 방향이 제한되는 상태.

- 2링크 로봇에서: θ₂ = 0° 또는 180° 근처 (링크가 완전히 펼쳐지거나 완전히 접힘)
- 이 상태에서는 일부 방향으로 끝점을 움직이기 어렵다.
- 실습 슬라이더에서 직접 체험해볼 수 있다.

### 선수 지식

- 링크와 관절 (`robot.struct.link-joint`)
- 정기구학 (`robot.kinematics.fk`)
