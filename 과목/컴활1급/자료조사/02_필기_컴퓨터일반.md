### 4-2. 하드웨어

| 항목 | 세부 개념 |
|------|----------|
| CPU 구성 | 제어장치(CU), 연산장치(ALU), 레지스터(가장 빠른 저장소) |
| 메모리 계층 | 레지스터 > 캐시메모리 > 주기억장치(RAM) > 보조기억장치 |

| DRAM vs SRAM | DRAM: 콘덴서, 재충전 필요, 일반 메모리 / SRAM: 트랜지스터, 캐시용, 고속 |
| 보조기억장치 | HDD(자기, 회전체), SSD(반도체, 충격 강함·소음 없음), CD/DVD/BD |
| 입력장치 | 키보드, 마우스, 스캐너, 디지타이저, 터치스크린, OMR·OCR·MICR·바코드리더 |
| 출력장치 | 모니터(해상도·재생률), 프린터(레이저·잉크젯·도트), 플로터 |
| RAID | RAID 0(스트라이핑), RAID 1(미러링), RAID 5(분산 패리티) |
| 인터럽트 | 외부·내부·소프트웨어 인터럽트, 채널(I/O 전담 프로세서) |
| 포트 | USB, HDMI, PS/2, IEEE 1394, Thunderbolt |
| BIOS & CMOS | BIOS(펌웨어, 부팅 초기화), CMOS(설정값 저장, 배터리 지원) |
| 마이크로프로세서 | 클록 주파수(높을수록 빠름), 코어 수, 캐시 크기 |
| 업그레이드/파티션 | 하드웨어 교체·추가, FAT32·NTFS 파티션 |
| PC 응급처치 | 블루스크린, 부팅 오류, 디스크 오류 점검 |

## 6. 자주 틀리는 함정 포인트

### 운영체제 / Windows
- **NTFS 파일 시스템**: 최대 볼륨 256TB, 파일명 255자, 유니코드 지원, FAT32보다 보안·안정성·성능 우수 — FAT32와 수치 혼동 주의
- **휴지통 미사용 케이스**: 플로피디스크·USB·DOS 명령 프롬프트·네트워크 드라이브에서 삭제 시 → 휴지통 거치지 않고 즉시 삭제
- **바로 가기 아이콘**: 원본 삭제해도 바로 가기 아이콘 남음 / 반대로 바로 가기 삭제해도 원본 유지
- **레지스트리 편집기**: Regedit.exe (Regedit32.exe 아님)
- **장치 관리자 아이콘**: 아래 화살표↓=사용 안 함, 물음표?=미설치, 느낌표!=오작동 — 물음표/느낌표 혼동 주의
- **선점형 멀티태스킹**: OS가 강제 제어 → 프리즈 프로그램 강제 종료 가능 (비선점형과 반드시 구분)

### 하드웨어
- **캐시메모리**: 주기억장치와 CPU 사이 고속 버퍼, SRAM 사용
- **DRAM vs SRAM**: DRAM=재충전 필요·대용량·저가, SRAM=재충전 불필요·소용량·고가·캐시용
- **RAID 0**: 속도 향상, 안정성 없음 (패리티 없음)
- **USB 포트**: 직렬 인터페이스 (병렬 아님)

### 정보통신·인터넷
- **OSI 7계층 장비 매핑**: 물리층=리피터·허브, 데이터링크층=브리지·스위치, 네트워크층=라우터, 전송층=게이트웨이 — 계층별 장비 혼동 빈출
- **IPv6**: 128비트, 16진수 4자리 8그룹 콜론(:) 구분 — IPv4(32비트, 10진수 점 구분)와 혼동
- **FTP 포트**: 21번(제어), 20번(데이터) — 포트 번호 반드시 구분
- **링형 토폴로지**: 한 노드 고장 → 전체 통신 마비 (버스형과 혼동)

### 보안
- **비밀키 vs 공개키**: 비밀키(대칭키, 빠름, 키 배송 문제), 공개키(비대칭키, 느림, 키 배송 문제 없음)
- **바이러스 vs 웜**: 바이러스=숙주 프로그램 필요·자기복제, 웜=독립 실행·네트워크 자가 전파
- **DDoS vs DoS**: DoS=단일 공격자, DDoS=다수 분산 공격

### 멀티미디어
- **GIF**: 256색(8비트), 무손실, 애니메이션 지원 → JPEG(손실)와 특성 혼동
- **벡터 방식**: 확대해도 품질 저하 없음 (비트맵과 반대)
- **MPEG-2**: DVD·HDTV용 (MPEG-4를 인터넷·모바일로 혼동하지 않기)
- **WAV**: 비압축 고음질, 용량 큼 (MP3와 혼동)

### 소프트웨어
- **컴파일러 vs 인터프리터**: 컴파일러=전체 번역·실행 파일 생성·실행 빠름, 인터프리터=줄 단위·실행 느림·대화형
- **프리웨어 vs 쉐어웨어**: 프리웨어=무료·기간 제한 없음, 쉐어웨어=일정 기간 무료 후 유료
