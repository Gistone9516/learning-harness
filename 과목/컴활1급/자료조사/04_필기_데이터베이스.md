## 2. 단원별 출제 경향 및 함정 포인트

### 대단원 I — 이론 핵심 (매 회 3~5문항)

| 주제 | 경향·함정 |
|------|----------|
| **키 종류** | 후보키 ≠ 기본키, 슈퍼키·대체키 혼동 자주 출제. "최소성"이 있으면 후보키, 없으면 슈퍼키. |
| **무결성** | 개체 무결성 = 기본키는 NULL 불가·중복 불가. 참조 무결성 = 외래키는 NULL이거나 참조 테이블 기본키 값과 일치. "참조 무결성 위반 사례" 선지 함정. |
| **정규화** | 2NF = 부분 함수 종속 제거, 3NF = 이행 함수 종속 제거, BCNF = 결정자가 후보키. 단계 순서·조건 오답 선지 빈출. |
| **E-R 다이어그램** | 개체(직사각형)·속성(타원)·관계(마름모) 기호 구분. 카디널리티(1:1 / 1:N / M:N) 혼동 주의. |
| **스키마** | 3단계 스키마 각 레벨의 독립성(논리적·물리적) 명칭 오답 함정. |
| **DBMS 언어** | DDL = CREATE·DROP·ALTER / DML = SELECT·INSERT·UPDATE·DELETE / DCL = GRANT·REVOKE 구분. |

### 대단원 II — 테이블 (매 회 2~3문항)

| 주제 | 경향·함정 |
|------|----------|
| **데이터 형식** | "자동 번호"는 기본키로만 쓸 수 있다는 오답 함정(사실은 아님). 날짜/시간 형식 코드(yyyy/mm/dd 등) 입력 마스크 기호 혼동. |
| **입력 마스크** | 0(필수 숫자) vs 9(선택 숫자), L(필수 문자) vs ?(선택 문자), < (소문자 변환) vs >(대문자 변환). |
| **관계 설정** | 참조 무결성 체크 시 "연쇄 업데이트"·"연쇄 삭제" 체크 여부와 실제 동작 결과 오답 선지. |
| **색인** | "예(중복 불가)"는 고유 인덱스, 기본키는 자동 고유 색인. |

### 대단원 III — 쿼리·SQL (매 회 5~7문항, 비중 최대)

| 주제 | 경향·함정 |
|------|----------|
| **SQL 실행 순서** | FROM → WHERE → GROUP BY → HAVING → SELECT → ORDER BY 논리 순서. HAVING 조건에 집계 함수 사용 가능 vs WHERE는 불가 — 자주 오답 선지. |
| **크로스탭 쿼리** | 행 머리글·열 머리글·값 역할 구분. TRANSFORM~PIVOT 구문. |
| **실행 쿼리** | 추가(INSERT INTO~SELECT) / 삭제(DELETE) / 업데이트(UPDATE~SET) / 테이블 만들기(SELECT~INTO) — 실행 전 경고 메시지 여부도 출제. |
| **조인** | 내부 조인 = 공통 값만 / 왼쪽 외부 조인 = 왼쪽 테이블 전체. NULL 포함 결과 행 수 계산 함정. |
| **IIF 함수** | IIF(조건, 참값, 거짓값) 중첩 사용 결과 오답 선지. |
| **Like 연산자** | * = 여러 문자, ? = 한 문자, # = 한 숫자. |

### 대단원 IV — 폼·컨트롤 (매 회 3~4문항)

| 주제 | 경향·함정 |
|------|----------|
| **컨트롤 구분** | 바운드 = 테이블/쿼리 필드 연결, 언바운드 = 독립, 계산 = 식으로 계산. 레이블은 언바운드 — 혼동 주의. |
| **이벤트 순서** | BeforeUpdate → AfterUpdate, OnOpen → OnLoad 순서 오답 선지. |
| **콤보박스 vs 목록 상자** | 콤보박스 = 직접 입력 가능, 목록 상자 = 목록에서만 선택. |
| **하위 폼** | 연결 필드(기본 폼 연결 필드 ↔ 하위 폼 연결 필드) 설정 개념. |
| **조건부 서식** | 식이 참인 경우 서식 적용 — 조건 작성 방식 오답 선지. |

### 대단원 V — 보고서 (매 회 2~3문항)

| 주제 | 경향·함정 |
|------|----------|
| **구역 종류** | 7개 구역(보고서 머리글·페이지 머리글·그룹 머리글·본문·그룹 바닥글·페이지 바닥글·보고서 바닥글) 각각의 역할·출력 위치. |
| **그룹 집계** | 그룹 바닥글에서만 그룹 소계 표시 가능. 보고서 바닥글 = 전체 합계. |
| **페이지 번호** | =[Page] & "/" & [Pages] 형식 구분. |

### 대단원 VI — 매크로·VBA (매 회 1~2문항)

| 주제 | 경향·함정 |
|------|----------|
| **매크로 함수 혼동** | OpenForm vs OpenReport 인수 구조(폼명·보기형식·조건·데이터 모드) 오답 선지. |
| **이벤트 매크로 vs 독립 매크로** | 이벤트 매크로 = 특정 컨트롤 이벤트에 연결 / 독립 매크로 = 탐색 창에서 직접 실행. |
| **VBA 기초** | If~ElseIf~End If, For~Next, Do While~Loop 기본 구문 오류 선지. |
| **DoCmd.OpenForm** | 조건(WHERE 절) 인수 위치·형식 오답 선지. |
